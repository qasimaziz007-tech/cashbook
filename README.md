# Esthetics Auto CashBook

A complete cash book management system for automotive businesses built with HTML, CSS, and JavaScript.

## Features

- 💰 **Transaction Management**: Track income and expenses with detailed categorization
- 👥 **Employee Management**: Manage staff information, payroll, and attendance
- 🚗 **Vehicle Reports**: Generate detailed vehicle-specific financial reports
- 📊 **Dashboard**: Real-time financial summaries and analytics
- 📈 **Export Functions**: Export data to Excel and PDF formats
- 🔐 **User Management**: Multi-user support with admin controls
- 💼 **Payroll System**: Complete salary calculation and slip generation
- 📅 **Attendance Tracking**: Monitor employee attendance and leaves
- 💸 **Advance Management**: Track employee advances and deductions

## Technology Stack

- Pure HTML5, CSS3, and JavaScript
- Local Storage for data persistence
- Chart.js for data visualization
- jsPDF for PDF generation
- SheetJS for Excel export/import
- Font Awesome for icons

## Setup

1. Clone this repository
2. Open `index.html` in any modern web browser
3. Login with admin credentials
4. Start managing your automotive business finances!
