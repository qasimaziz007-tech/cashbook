# 📦 Esthetics Auto CashBook - Installer Packages

Professional CashBook Management System - Ready to Install on Any Platform!

## 🚀 Available Downloads

### 🍎 **macOS** (77KB)
**File**: `Esthetics-Auto-CashBook-1.0.0-Mac.dmg`
```
1. Double-click the .dmg file
2. Drag "Esthetics Auto CashBook" to Applications folder  
3. Launch from Applications or Launchpad
4. Login: admin / admin123
```

### 🖥️ **Windows** (47KB)  
**File**: `Esthetics-Auto-CashBook-1.0.0-Windows.zip`
```
1. Extract the ZIP file to any folder
2. Double-click "CashBook.vbs" to launch silently
3. Or double-click "CashBook.bat" for console version
4. Login: admin / admin123
```

### 🐧 **Linux** (48KB)
**File**: `esthetics-auto-cashbook_1.0.0_linux.tar.gz`
```
1. Extract: tar -xzf esthetics-auto-cashbook_1.0.0_linux.tar.gz
2. Install: sudo mv esthetics-auto-cashbook_1.0.0_all /opt/
3. Create symlink: sudo ln -s /opt/.../usr/bin/esthetics-auto-cashbook /usr/local/bin/
4. Run: esthetics-auto-cashbook
```
